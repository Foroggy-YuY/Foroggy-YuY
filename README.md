### 🍄 What's in the workshop

> *All repos are private for now. I'll open them when they stop being embarrassing. Estimated time: unknown.*

&nbsp;&nbsp;**[Airy](https://github.com/Foroggy-YuY)** — A local AI voice assistant with an actual personality. Runs fully offline, has a Big Five personality system, and Constitutional AI-inspired safety logic. 40k+ lines. Still talking back.

&nbsp;&nbsp;**[Ribbit OS](https://github.com/Foroggy-YuY)** — A Linux distro built around the idea of a "small, warm home". Meets you kindly, teaches you the terminal, and when you're ready for Arch — it lets you go. Currently exists mostly as a vision document and a lot of feelings.

&nbsp;&nbsp;**[AiryVeil](https://github.com/Foroggy-YuY)** — A password manager in Rust. Post-quantum encryption, Argon2id, steganography features I added because I could. 179+ tests. The borrow checker and I have an understanding now.
